Install Oversight protection
  1. Tap Install Oversight protection above.
  2. Open Settings — you'll see Profile Downloaded near the top.
  3. Tap it, then Install, and enter this device's passcode.
  4. Done — web filtering and Safe DNS are active.

Download Windows helper (.bat)

Download Linux helper (.sh)

Run sudo bash oversight-linux-dns.sh. Or set DNS to 1.1.1.3 / 1.0.0.3 in your network settings.

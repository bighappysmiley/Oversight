Set up protection

Enter the setup code from your parent dashboard to protect this device.

Install Oversight protection
  1. Tap Install Oversight protection above.
  2. iOS will say a profile was downloaded. Open Settings.
  3. Tap Profile Downloaded near the top, then Install.
  4. Enter this device's passcode and confirm. Protection is now active.

To remove it later, your parent's protection password is required.

Download Oversight app

Your setup code:

--------
  1. Tap Download Oversight app and open the downloaded file.
  2. If asked, allow installing apps from this browser.
  3. Open Oversight, enter the setup code above, and set a name.
  4. Approve device administrator access when prompted.

Uninstalling later requires your parent's protection password.
